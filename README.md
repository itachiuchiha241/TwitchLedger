🎮 TwitchLedger

A modern React dashboard for tracking Twitch support across your favorite creators.

✨ Features
🔐 Twitch OAuth Login
👤 Twitch User Dashboard
💜 Gifted Subscription Tracker
💎 Bits Donation Tracker
📊 Badge Progress System
🌙 Dark / Light Theme
🔎 Creator Search
📺 Supported Creators Page
🎨 Modern Responsive UI
----------------------------------------------------------------------------------------------------------------------------

🛠️ Built With
React
Vite
JavaScript
CSS3
Twitch OAuth API

----------------------------------------------------------------------------------------------------------------------------

🚀 Getting Started

Clone the repository:
git clone https://github.com/itachiuchiha241/TwitchLedger.git
Install dependencies:
npm install
Start the development server:
npm run dev
